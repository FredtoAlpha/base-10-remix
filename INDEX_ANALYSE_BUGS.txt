LES 4 BUGS EN UNE PHRASE CHACUN
================================

Bug 1 : CSS header.hidden n'a pas "display: none", donc le header prend toujours de la place
Bug 2 : Classes Tailwind statiques (lg:grid-cols-4) forcent un layout 1x4 au lieu de s'adapter
Bug 3 : state.students n'est pas reinitialise entre regroupements, donc les eleves se melangent
Bug 4 : Phase 2 et 3 font la meme chose, 150 lignes de code duplique, fusionnables

QUESTIONS FREQUENTES
====================

Q: Lesquels sont CRITIQUES ?
A: Bug 3 (donnees qui se melangent) est CRITIQUE
   Bug 1 et 2 sont importants mais cosmétiques
   Bug 4 est optimisation

Q: Je peux ignorer certains bugs ?
A: Bug 3 : NON, données incorrectes
   Bug 1 : NON, UI non-fonctionnelle
   Bug 2 : NON, Layout cassé
   Bug 4 : OUI, juste optimisation

Q: Combien de temps pour tout fixer ?
A: 1 minute pour bugs 1,2,3
   30-45 minutes pour bug 4
   Test inclus

Q: Risque de tout casser ?
A: NON, changements très localisés et sûrs
   Aucune modification de logique métier
   Juste CSS et nettoyage d'état
